Go学习

环境：

操作系统：mac

安装

  1. 去官网地址下载mac版本的安装包，双击按照提示安装就行了
  2. 打开terminal，输入go version，验证是否安装成功
  3. 输入cd /usr/local/go查看go的安装目录

开始使用

  1. 创建一个文件夹hello，然后就进入文件夹（我是在桌面创建的）

  2. 开启依赖跟踪和管理，go mod init example/hello,运行成功后会在hello目录下有一个go.mod文件

  3. 通过编辑器（我使用的vscode）创建一个hello.go文件

  4. 在hello.go输入一下代码

    package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

  5. 执行go run .命令，控制台打出Hello, World!

模块

创建模块

  1. 创建一个文件夹greet，然后进入文件夹

  2. 开启依赖跟踪和管理，go mod init example.io/greetings,执行成功后greet目录下有个go.mod文件

  3. 创建greetings.go文件

  4. 输入一下代码

    //声明一个名为greetings的包，组织相关函数
package greetings

import "fmt"

//实现Hello函数
//Hello：函数名，大写字母开头表示是一个Exported Name，能被包外调用，否则不能被包外调用
//name：入参
//第一个string：入参类型
//第二个string：出参类型
func Hello(name string) string{
    //message 返回参数名称
  //:= 在go中是简写，
  //等同于 var message string
  //message = fmt.Sprintf("Hi, %v. Welcome!", name)
    message := fmt.Sprint("Hi, %v. Welcome!", name)
  //返回值
    return message
}

调用模块

  1. 在hello文件夹下，创建一个hello.go文件

  2. 输入以下代码

    package main

import (
    "fmt"
    "example.io/greetings"
)

func main(){
    message := greetings.Hello("from hello.go file invoke")
    fmt.Println(message)
}

  3. 运行命令go mod edit -replace example.com/greetings=../greetings,关联导入"example.io/greetings"的实际地址

  4. 运行命令go mod tidy,同步添加example.io/greetings模块的依赖项

  5. 运行命令go run .，得到结果Hi, from hello.go file invoke. Welcome!

错误处理

  1. 打开greetings/greetings.go文件，进行如下修改：

    package greetings

import (
    "fmt"
  //导入errors库
    "errors"
)

//多了一个error返回参数，注意go函数可以返回多个参数
func Hello(name string) (string, error){
  //如果name为空，返回error类型
    if name == "" {
        return "", errors.New("empty name")
    }


    message := fmt.Sprintf("Hi, %v. Welcome!", name)
  //如果不为空，error位置返回一个nill，用于调用方判断
    return message, nil
}

  2. 打开hello/hello.go文件，进行如下修改：

    package main

import (
  //导入日志库
    "log"
    "fmt"
    "example.com/greetings"
)

func main(){

    //设置log参数
    log.SetPrefix("greetings: ")
  //0 不打印日期参数 
    log.SetFlags(0)

    //返回俩参数，用俩参数接收
    message,err := greetings.Hello("")
    //这里对greetings返回的err参数进行判断
  //如果是nil代表没有错误，继续正确流程；不是nil进行一个错误的打印，然后退出程序
    if(err != nil){
        log.Fatal(err)
    }

    fmt.Println(message)
}

  3. 进入hello目录，执行go run . ，打印如下：

    greetings: empty name
exit status 1

    总结，go中的通用错误处理就是这样，返回一个error作为参数，然后调用方进行检查

返回随机问候

为了完成这个功能，需要使用slice结构，这个结构大致可以看作Java中的ArrayList，可以动态扩容。

修改greetings/greetings.go文件，改动如下：

package greetings

import (
    "errors"
    "fmt"
    "math/rand"
    "time"
)

func Hello(name string) (string, error){
    if name == "" {
        return "", errors.New("empty name")
    }

    message := fmt.Sprintf(randomFormat(), name)
    return message, nil
}

//init方法会在程序启动全局变量初始化后自动被调用
func init() {
  //设置随机数种子
    rand.Seed(time.Now().UnixNano())
}

//随机返回一个greet模版
func randomFormat() string{
    formats := []string{
        "Hi, %v. Welcome!",
        "Great to see you, %v!",
        "Hail, %v! Well met!",
    }
    return formats[rand.Intn(len(formats))]
}

执行之后，结果如下：

knight@knightdeMBP hello % go run .
Hi, jj. Welcome!
knight@knightdeMBP hello % go run .
Great to see you, jj!
knight@knightdeMBP hello % go run .
Hi, jj. Welcome!

返回多个人的问候

由于之前greetings.go中的Hello函数入参只有一个，我们现在需要多个。

go不支持函数重载，只有新写一个函数，具体代码如下：

package greetings

import (
    "errors"
    "fmt"
    "math/rand"
    "time"
)

func Hello(name string) (string, error){
    if name == "" {
        return "", errors.New("empty name")
    }
    message := fmt.Sprintf(randomFormat(), name)
    return message, nil
}

//入参是多个参数，返回一个map结构，一个error参数
func Hellos(names []string) (map[string]string, error){
  //初始化map结构
    messages := make(map[string]string)

  //遍历入参，index代表names的索引，name代表索引位置对应的值
  //如果不关心索引的话，index可以用 '_'代替，表示忽略索引参数，例如：for _,name := range namse{}
    for index, name := range names{
        fmt.Println(index)
        message, err := Hello(name)
    if (err != nil) {
      return nil, err
    }
    //设置map
        messages[name] = message
    }
    return messages, nil
}


func init() {
    rand.Seed(time.Now().UnixNano())
}

func randomFormat() string{
    formats := []string{
        "Hi, %v. Welcome!",
        "Great to see you, %v!",
        "Hail, %v! Well met!",
    }
    return formats[rand.Intn(len(formats))]
}

然后对调用方进行修改，具体如下：

package main

import (
    "log"
    "fmt"
    "example.com/greetings"
)

func main(){

    //设置log参数
    log.SetPrefix("greetings: ")
    log.SetFlags(0)

  //创建一个names的slice结构
    names := []string{"zhangsan", "lisi", "wangwu"}
  //作为入参，然后接一个map
    message,err := greetings.Hellos(names)
    if(err != nil){
        log.Fatal(err)
    }
  //打印
    fmt.Println(message)
}

到hello目录下，执行命令go run .，得到结果：

knight@knightdeMacBook-Pro hello % go run .
map[lisi:Hi, lisi. Welcome! wangwu:Great to see you, wangwu! zhangsan:Hi, zhangsan. Welcome!]

单元测试

  1. 在greetings目录创建一个greetings_test.go文件

  2. 往greetings_test.go文件写入如下代码

    
package greetings

import (
    "testing"
    "regexp"
)

//创建一个测试单个名称的测试用例，测试类必须以TestName的形式，其中Name代表确切的测试过程
//指向 *testing.T 的指针，用于报告，或者记录你的测试过程
func TestHelloName(t *testing.T)  {
    name := "Gladys"
    want := regexp.MustCompile(`\b`+name+`\b`)
  //正常调用greetings.go里面的Hello方法
    msg, err := Hello("Gladys")
  //如果返回的参数，包含Gladys，测试通过，否者打印日志
    if( !want.MatchString(msg) || err!=nil ){
        t.Fatalf(`Hello("Gladys") = %q, %v, want match for %#q, nil`, msg, err, want)
    }
}
//创建一个测试空名称的测试用例
func TestHelloEmpty(t *testing.T){
    msg, err := Hello("")
    if msg != "" || err == nil {
        t.Fatalf(`Hello("") = %q, %v, want "", error`, msg, err)
    }
}

  3. 在当前目录，即greetings目录下，打开命令输入go test或者go test -v

  4. 修改greetings.Hello方法，重复步骤3，来查看失败的测试结果

编译安装

本主题主要学习两个命令,go build：编译包和他们的依赖，不会安装结果，go install：编译并安装结果

  1. 进入hello目录，执行go build命令，会在当前目录下看到一个hello可执行文件，执行./hello查看结果

  2. 发现go的安装目录，执行go list -f '{{.Target}}'，看到/Users/knight/go/bin/hello，这是我的机器上默认的安装路径

  3. 修改安装路径，两种方式，如下：

    //方式一
//mac 或者linux
export PATH=$PATH:/path/to/your/install/directory

//方式二
go env -w GOBIN=/path/to/your/bin

    我这里用方式二设置的目录

  4. 安装， 执行go install,然后就看到在我们上面设置的目录里面有一个hello的可执行文件了
